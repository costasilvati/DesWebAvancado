Capítulo 5 Lista de Atividades — Estudo JavaScript (NodeJS)

Disciplina: Desenvolvimento Web Avançado
Objetivo: Reforçar os conceitos básicos de JavaScript aplicados ao desenvolvimento back-end

5.1 Instruções

  • Desenvolver os exercícios em Node.js
  • Utilizar console.log() para exibir resultados
  • Criar um único arquivo lista_revisao.js ou arquivos separados
  • Não utilizar bibliotecas externas

  1. Declare três variáveis: nome, idade e curso. Exiba uma frase com template string.

  2. Dado um número, verifique se ele é:

    • positivo
    • negativo
    • zero

  3. Crie um programa que calcule a média de três notas e informe:

    • aprovado (>= 7)
    • reprovado (< 7)

  4. Verifique se uma pessoa pode dirigir considerando:

    • idade >= 18
    • possui carteira (true/false)

  1. Crie um array com 5 números e exiba todos usando for...of.

  2. Calcule a soma dos valores de um array manualmente (sem reduce).

  3. Conte quantos números são pares em um array.

  4. Dado um array de números, crie outro array contendo apenas os números maiores que 10.

  1. Crie um objeto produto com:
    • nome
    • preço
    • quantidade

Exiba o valor total em estoque.

  1. Crie um array de objetos representando alunos:
{ nome: "Ana", notas: [7,8,9] }

Exiba o nome de cada aluno.

  1. Para cada aluno, calcule a média das notas.

  1. Crie uma função tradicional que calcule a média de um array.

  2. Reescreva a função usando arrow function.

  3. Crie uma função que receba um produto e retorne:

  • “estoque baixo” (quantidade < 5)
  • “estoque ok”

  1. Dado um array de números, utilize map para gerar o dobro de cada valor.

  2. Utilize filter para retornar apenas números pares.

  3. Utilize reduce para calcular a soma de um array.

  1. Dado um array de produtos:
{ nome: "Notebook", preco: 2000 }

Calcule o valor total usando reduce.

  1. Dado um array de alunos:
{ nome: "Ana", notas: [7,8,9] }

Crie um novo array contendo:

{ nome: "Ana", media: X }
  1. Filtre apenas alunos com média >= 7.
  1. Gere uma lista apenas com os nomes dos aprovados.
  1. Crie um sistema simples de produtos:
{ nome, preco, quantidade }

O programa deve:

  • exibir todos os produtos
  • calcular valor total de cada produto
  • identificar produtos com estoque baixo
  • calcular valor total do estoque
  • encontrar o produto mais caro
  1. Dado um array de usuários:
{ nome, idade, ativo }

O programa deve:

  • listar usuários ativos
  • contar quantos estão ativos
  • calcular média de idade
  • listar apenas maiores de idade
  1. Crie um sistema de pedidos:
{ cliente, total, status }

O programa deve:

  • listar pedidos aprovados
  • calcular total de vendas
  • agrupar por cliente
  • identificar cliente que mais comprou

5.2 Dica importante

Se você não souber por onde começar:

  1. Resolva com for primeiro
  2. Depois refatore usando map, filter e reduce
