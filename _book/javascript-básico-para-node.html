Disciplina: Desenvolvimento Web Avançado

15.1 1. O que é JavaScript no contexto do Node.js

JavaScript é uma linguagem de programação originalmente criada para execução em navegadores.
Com o Node.js, o JavaScript também pode ser executado no servidor, permitindo o desenvolvimento de aplicações back-end.

Em Node.js, scripts JavaScript são executados diretamente pelo interpretador:

node arquivo.js

Exemplo simples:

console.log("Hello Node.js");

15.2 2. Declaração de Variáveis

JavaScript possui três formas principais de declarar variáveis:

Palavra-chave Escopo Observação
var função legado, evitar em código moderno
let bloco variável mutável
const bloco valor constante

Exemplo:

let nome = "Ana";
const idade = 20;

console.log(nome);
console.log(idade);

Boa prática:

  • usar const por padrão
  • usar let quando o valor precisa mudar

15.3 3. Tipos de Dados

JavaScript é dinamicamente tipada, ou seja, o tipo é definido em tempo de execução.

Tipos primitivos principais:

Tipo Exemplo
string "Olá"
number 10, 3.14
boolean true, false
null ausência de valor
undefined variável declarada sem valor

Exemplo:

let nome = "Carlos";
let idade = 30;
let ativo = true;

console.log(typeof nome);
console.log(typeof idade);
console.log(typeof ativo);

15.4 4. Operadores Matemáticos

Operador Descrição
+ soma
- subtração
* multiplicação
/ divisão
% resto da divisão
** potência

Exemplo:

let a = 10;
let b = 3;

console.log(a + b);
console.log(a % b);
console.log(a ** 2);

15.5 5. Operadores Relacionais

Operador Significado
> maior
< menor
>= maior ou igual
<= menor ou igual
== igualdade (com coerção de tipo)
=== igualdade estrita

Exemplo:

let x = 10;
let y = "10";

console.log(x == y);
console.log(x === y);

Boa prática: usar sempre ===.

15.6 6. Operadores Lógicos

Operador Significado
&& AND
| | OR
! NOT

Exemplo:

let idade = 20;
let possuiCarteira = true;

if (idade >= 18 && possuiCarteira) {
    console.log("Pode dirigir");
}

15.7 7. Estruturas Condicionais

15.7.1 if / else

let idade = 18;

if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

15.7.2 switch

let dia = 2;

switch(dia){
    case 1:
        console.log("Segunda");
        break;
    case 2:
        console.log("Terça");
        break;
    default:
        console.log("Outro dia");
}

15.8 8. Estruturas de Repetição

15.8.1 for

for (let i = 0; i < 5; i++) {
    console.log(i);
}

15.8.2 while

let i = 0;

while (i < 5) {
    console.log(i);
    i++;
}

15.8.3 for…of (muito comum em JavaScript)

let numeros = [1,2,3,4];

for (let n of numeros) {
    console.log(n);
}

15.9 9. Funções

Funções podem ser declaradas de várias formas.

15.9.1 Função tradicional

function soma(a, b){
    return a + b;
}

console.log(soma(5,3));

15.9.2 Arrow Function

Muito utilizada em aplicações Node.js modernas.

const soma = (a, b) => {
    return a + b;
};

Versão reduzida:

const soma = (a,b) => a + b;

Leia mais sobre arrow functions

15.10 10. Arrays

Arrays armazenam coleções de valores.

let numeros = [10, 20, 30];

console.log(numeros[0]);

Iteração:

for (let n of numeros) {
    console.log(n);
}

15.11 11. Objetos

Objetos são estruturas chave-valor.

let usuario = {
    nome: "Ana",
    idade: 25,
    ativo: true
};

console.log(usuario.nome);

Iteração:

for (let chave in usuario){
    console.log(chave, usuario[chave]);
}

15.12 12. Template Strings

Permitem interpolação de variáveis.

let nome = "João";
let idade = 30;

console.log(`Nome: ${nome}, Idade: ${idade}`);

15.13 13. Módulos no Node.js

Node.js utiliza módulos para organizar código.

15.14 Exportando

function soma(a,b){
    return a+b;
}

module.exports = soma;

15.14.1 Importando

const soma = require('./soma');

console.log(soma(2,3));

15.15 14. Boas práticas iniciais

  • utilizar const sempre que possível
  • preferir === em vez de ==
  • usar arrow functions
  • manter funções pequenas
  • organizar código em módulos

15.16 15. Executando programas Node.js

Estrutura típica:

projeto/
 ├── index.js
 ├── utils.js
 └── package.json

Executar:

node index.js
